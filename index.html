SMART SYSTEM FOR OPTIMIZED ORGANIC CROP ROTATION USING PRECISION AGRICULTURE DATA

Sri Lanka’s first IOT and Machine Learning Based Smart System for Crop Rotation planning with real time expert advices.

Research Scope

Liter​ature Survey

Hearing impairment or Hearing Impaired (HI) is defined as the inability of an individual to hear sounds. This includes people with any degree of hearing loss completely by birth or later life. According to the statistiscs privided by The World Health Organization 9% of Sri Lankans suffer from hearing loss while 15% of them are completely deaf. The main communication method used by the HI is known as Sign language.

Sign language is a means of communication using hand gestures and body movements. Among many Variations of sign languages used around the world Sri Lankan Sign Language is the dialect of sign language used by the hearing impaired Community of Sri Lanka for communication.HI community of Sri Lanka face challenges such as limited access to tools to study and communication issues. Also, when considering the Sri Lankan deaf community, there are very limited number of research and platforms available specifically focusing on them. The HI of Sri Lanka are unable to use platforms or applications which are based on other sign languages and face issues using regular systems for non-hearing-impaired people.

Research Gap

In relation to the literature surveys that were conducted for the module of text-to-sign language conversion, few research gaps were discovered and listed below:

  •  There are no any system that address the organic crop rotation planning in Sri Lankan context.
  •  Existing platforms doesn't take the measured data for the crop rotation plan.
  •  Usage of providing organic and nonchemical solutions for the identified problems.

Research Problem

  • ►
    No online platforms that teaches SSL and also tracks student's motions and provide feedbacks.
  • ►
    No online platforms that converts SSL to ASL.
  • ►
    No platform that provides HI to search using sign language and retrieve information using google.
  • ►
    No platform tha captions youtube videos in SSL along with emotion analysis.

Objectives

The main objective of this system is to provide the opportunity for the stakeholders to create their crop rotation plan with the help analysis of soil, climate and pests.

Sub Objectives

  •  Gather soil data using IoT device and use it as parameters in an algorithm.
  •  Identify the pests and diseases based on the provided image and provide solutions through an algorithm.
  •  Gather climate data using IoT device and use it as a parameters in an algorithm.

Methodology

Teaching SSL and Ev​aluati​on

  • Display words and relevant SSL signs using an animated avatar gif.
  • Evaluate user input signs using Media Pipe Holistic and Tenser-flow object detection model.
  • Correct answer is compared with the detected answer saved in the pickle file from the trained sequential neural network.

SSL to ASL Translation

  • Convert a given gesture in Sri Lankan Sign Language into text.
  • Convert the recognized text into the corresponding gesture in American Sign Language.
  • Display translated ASL signs using an animated avatar gif.

Youtube Video Captioning Module

  • Download video using the user provided link.
  • Convert transcript of the video to SSL.
  • Identifying emotion using text, speech and facial expressions denoted in the video.
  • Map releveant converted signs and emotions and display in SSL.

Google Search Module

  • Conversion of Sri Lankan Sign Language to text.
  • Converted text is used for the process of Google Search, and BeautifulSoup is used for Web Scraping.
  • Convert search results to SSL and display to the user.
